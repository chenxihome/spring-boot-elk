# spring-boot-elk
基于springboot开发的自主日志采集系统
*****************************************************************
本示例主要基于springboot对logAppender进行二次开发，集中采集应用系统日志信息，通过
kafka消息中间件写入ElasticSearch中，可通过kibana进行日志的分析和查询
